Administrador de Pacientes de Veterinaria - APV

MongoDB
Express.js
ReactJS
Node.js
Tailwind CSS

Un proyecto desarrollado bajo el stack de MERN. En la parte del back-end la API fue desarrollada con Node.js y Express.js, cuenta con distintos endpoints para autenticación de usuarios, recuperación de passwords y validación de usuario nuevos a través de token enviado al correo registrado. La funcionalidad de envío de emails se la realiza con el paquete Nodemailer.

La autenticación se realiza a través de JWT (Json Web Token) y se gestiona con un middleware, de la misma manera cuenta con endpoints que abarcan todo el CRUD de pacientes de cada veterinario.

Para el desarrollo del frontend se utilizó ReactJS y tailwind con framework CSS.

El proyecto no está terminado. Aún se están implementando funcionalidades.

Es importante aclarar que el back-end de este proyecto está alojado en una instancia gratuita, por lo que la primera vez que se realizan peticiones suele demorar un poco. ¡Un poquito de paciencia que valdrá la pena!
